# dotfiles
My mac settings

## Procedure

```sh
// Install xcode
$ xcode-select install

// Install homebrew
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

// Install dotfiles
$ sh dotfiles.sh

// Change shell
$ chsh -s /usr/local/bin/fish
$ chsh -s /bin/zsh
```
